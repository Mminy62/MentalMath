# Mental Math - 암산 연습

경시 시험, 공무원 시험 혹은 두뇌 회전을 위한 간단한 암산 앱입니다 :)
공시생들의 20단까지의 단수 연습 앱이 없어 아쉽다는 의견에서 시작했습니다.

- 덧셈, 뺄셈, 나눗셈, 곱셈은 모두 자리수 선택 가능한 랜덤 문제를 제공
- 곱셈 메뉴에는 특정 단수 연습 가능 (2 ~ 20단 제공)
